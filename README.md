# MERN Notes App

A full-stack Note Taking application built with the **MERN Stack** (MongoDB, Express, React, Node.js). It features a modern, responsive UI styled with Tailwind CSS and DaisyUI, complete with real-time notifications and rate limiting handling.

 ## Features

* **Create Notes:** Add new notes with a Title and detailed Content.
* **Read Notes:** View all notes in a responsive grid layout.
* **View Details:** Click any note to read the full content in a dedicated view.
* **Edit Notes:** seamlessly Edit mode to update content.
* **Delete Notes:** Remove notes with safety confirmation dialogs.
* **Smart UI:**
    * Loading spinners for better user experience.
    * Toast notifications (Success/Error messages).
    * Rate Limit handling (Visual feedback when sending too many requests).
    * Responsive design for mobile and desktop.

##  Tech Stack

### Frontend
* **React:** UI Library
* **Tailwind CSS:** Utility-first CSS framework
* **DaisyUI:** Component library for Tailwind
* **Axios:** HTTP Client for API requests
* **React Router DOM:** Client-side routing
* **React Hot Toast:** Notification popups
* **Lucide React:** Icon set

### Backend
* **Node.js & Express:** Server environment
* **MongoDB & Mongoose:** Database and ODM
* **Cors:** Cross-Origin Resource Sharing handling
